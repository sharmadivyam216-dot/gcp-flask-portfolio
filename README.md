# Cloud & DevOps Portfolio Website

A modern cloud-native portfolio application built using Flask and deployed on Google Cloud Platform.

## Features

* Modern responsive UI
* Flask backend application
* Google Cloud App Engine deployment
* Cloud Build CI/CD integration
* Cloud & DevOps themed portfolio design
* Interactive animations and modern styling

## Tech Stack

* Python
* Flask
* HTML5
* CSS3
* Google Cloud Platform
* App Engine
* Cloud Build
* GitHub

## Running Locally

Clone the repository:

```bash
git clone https://github.com/sharmadivyam216-dot/gcp-flask-portfolio.git
```

Create virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python3 main.py
```
